Sobre o Projeto

Objetivo do Site

Este site foi desenvolvido como parte do bloco 3 do bootcamp da Generation Brasil. O principal objetivo é aplicar os conhecimentos adquiridos durante o treinamento, que incluem:

  • Desenvolvimento de páginas web responsivas
  • Implementação de conceitos de design e UX
  • Utilização de técnicas de HTML e CSS para criar layouts atraentes
  • Aplicação das melhores práticas de desenvolvimento web

Este projeto serve como um meio para consolidar as habilidades e conceitos aprendidos, garantindo uma compreensão prática e efetiva das ferramentas e técnicas usadas no desenvolvimento web.
